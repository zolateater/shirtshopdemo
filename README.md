# Описание

Демо проект на чистом JavaScript для демонстрации простого варианта использования WebGL.
Проект служит для шпаргалочных целей. Что внутри? Рисуем текстурка, наносим на модель.

# Как запустить -

    npm install

А затем

    gulp dev

# Как запустить тесты

Открыть build/TestRunner.html
